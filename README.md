## Usage

if initial run:

  - run 'npm install'
  - start MySQL server
  - run 'npm seed'

server start script is 'start'
react/webpack start script is 'build'
seeding script is 'seed'

home url path is 'localhost:9000'

## Requirements

- Node 10.13.0
- MySQL/Sequelize
- React
